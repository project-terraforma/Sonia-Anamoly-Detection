# Overture Maps Release Anomaly Analysis

Release Comparison: 2025-08-20.1 → 2025-09-24.0

1. Executive Summary

A systematic doubling of city features occurred across 15+ countries in the divisions theme, with 978 critical anomalies detected (99.5% of all issues). This appears to be a data processing error rather than legitimate geographic updates, given the consistent ~100% increase pattern and global scope.

2. Key Patterns Identified

  • Global City Duplication Pattern: 952 critical anomalies in divisions theme, all showing city feature counts doubling (~98-104% increases) across BR, CA, CN, EG, IN, IR, JP, MX, PH, PK, PT, RO, RU, US
  • Consistent Magnitude: All affected countries show nearly identical percentage increases (98-104%), suggesting systematic processing error
  • Geographic Scope: Affects major population centers globally, indicating upstream data pipeline issue
  • Single Geometry Degradation: One critical geometry length reduction in base/water/lake (-21.9%, ~730km loss)
  • Minimal Transportation Impact: Only 25 anomalies in transportation theme (2.5% of total)

3. Root Cause Hypotheses

  • Primary: Data deduplication failure in ETL pipeline causing duplicate city records to be ingested
  • Secondary: Source data provider introduced duplicate records upstream
  • Tertiary: Schema or ID generation changes causing existing features to be treated as new entities
  • Geometry Issue: Separate water feature processing error or coordinate system transformation problem

4. Recommended Actions

IMMEDIATE (Critical Priority):
  • Halt distribution of 2025-09-24.0 release for divisions theme
  • Investigate deduplication logic in city feature processing pipeline
  • Compare raw source data between releases to identify duplication point
SHORT-TERM (High Priority):
  • Implement automated detection for >50% feature count increases in future releases
  • Audit geometry processing pipeline for water features
  • Validate ID generation and feature matching algorithms
MEDIUM-TERM (Medium Priority):
  • Establish country-level feature count baselines with tolerance thresholds
  • Implement cross-theme anomaly correlation analysis
